Instalando o Docker
no macOS

Agora que você já verificou se seu computador atende aos requisitos para instalação do Docker, vamos ao passo a passo de como instalá-lo no sistema operacional macOS

Passo 1

Acesse o site oficial do Docker e clique no botão "Download Docker Desktop", ou em português, "Baixar Docker Desktop" que estará visível assim que você abrir o site, após isso, selecione a opção que seja compatível com seu chip(Intel ou Apple Silicon)

Passo 2

Após seguir o primeiro passo, você terá baixado o arquivo .dmg do Docker em seu computador, procure-o em seu explorador de arquivos, e ao encontrá-lo você deve o abrir e arrastar o ícone do Docker para a pasta aplications, agora, dentro desta pasta você irá abrir o Docker e dar as permissões necessárias, talvez o SO peça a senha de administrador. Depois de alguns segundos o ícone da baleia aparecerá na barra superior indicando que o Docker está rodando

Passo 3

Após seguir todos os passos, temos que testar para ver se está tudo certo, para fazer esse teste basta executar os dois códigos abaixo:


docker --version

docker run hello-world


Se a mensagem "Hello from Docker!" aparecer, o Docker foi instalado com sucesso e agora você pode desfrutar de tudo que o Docker oferece!
