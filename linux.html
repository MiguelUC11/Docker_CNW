Instalando o Docker
no Linux

Agora que você já verificou se seu computador atende aos requisitos para instalação do Docker, vamos ao passo a passo de como instalá-lo no sistema operacional Linux na distribuição Ubuntu que é a mais comum

Passo 1

Antes de tudo é necessário atualiazar os pacotes para suas versões mais recentes para evitar conflitos durante e após a instação, para fazer isso abra o terminar e execute o seguinte comando:


sudo apt update && sudo apt upgrade -y

Passo 2

Após a atualização dos pacotes também é necessário instalar dependências que possibilitarão a comunicação do sistema com o repositório do Docker. Execute o código abaixo para fazer isso:


sudo apt install apt-transport-https ca-certificates curl software-properties-common -y

Passo 3

Adicione a chave GPG oficial do Docker, essa chave usada para verificar se os pacotes instalados são originais e seguros, para adicionar essa chave basta você dar o seguinte comando:


curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo apt-key add -

Passo 4

Agora você deverá adicionar o repositório Docker para o sistema saber de onde baixar as versões mais recentes do Docker de forma segura, execute o seguinto comando no terminal para isso:


sudo add-apt-repository "deb [arch=amd64] https://download.docker.com/linux/ubuntu $(lsb_release -cs) stable"

Passo 5

Após realizar tudo isso, podemos finalmente instalar de fato o Docker, você só precisa executar esses dois comandos em sequência:


sudo apt update

sudo apt install docker-ce -y

Passo 6

Por último, só precisamos testar para ter certeza que o Docker foi instalado com sucesso em sua máquina. Vamos fazer isso executando os dois cógidos abaixo:


docker --version

docker run hello-world


Se a mensagem "Hello from Docker!" aparecer significa que ele foi instaldo com sucesso e que tudo ocorreu como deveria, agora você pode desfrutar de tudo que o Docker oferece!.
