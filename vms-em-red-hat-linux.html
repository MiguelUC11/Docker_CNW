Criando a máquina virtual com o Red Hat

Para iniciar você deve baixar uma imagen ISO do Red Hat Linux que pode ser encontrada aqui! Após isso você deve usar um software de virtualização, depois crie um servidor no software desejado para a criação da máquina virtual usando a ISO baixada do linux. Para a instalação é recomendado no mínimo 2 gigas de RAM, e 20 gigas de armazenamento para o Red Hat.

Instalador do Red Hat Linux

Ao iniciar a máquina, o instalador do Red Hat será carregado. O processo de instalação pede a escolha do idioma e do layout do teclado, além da configuração do disco de destino, da rede e do tipo de instalação, que pode ser mínima (sem interface gráfica) ou servidor com interface gráfica. Depois é só clicar em Begin Installation, definir a senha de root, criar um usuário normal e concluir o processo, reiniciando a máquina para começar a usar o sistema.
